# Student Lists

## Information

Student Lists is an Android application designed to manage and organize student information efficiently.

## Features

- Add, edit, and delete student details
- Works in all orientation
- Has Light/Dark mode color themes

## Tech Stack

- Java
- Android SDK
- SQLite
